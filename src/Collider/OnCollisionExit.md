# Collider.OnCollisionExit

当物体**脱离碰撞**另一个物体时被调用。


## 定义

```csharp
private void OnCollisionExit(Collision collision);
```

## 参数

| 参数名    | 类型      | 说明                                                         |
| --------- | --------- | ------------------------------------------------------------ |
| collision | Collision | 与该碰撞事件关联的 `Collision` 数据，有关 `Collision` 介绍见 [附录 #Collision](./appendix.md#collision) |
