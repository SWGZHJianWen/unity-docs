# Animator.SetBool

设置布尔参数。

## 定义

```csharp
public void SetBool(string name, bool value);
```

| 参数名 | 类型   | 必要 | 说明       |
|:------ |:------ |:---- |:---------- |
| name   | string | 是   | 参数名称   |
| value  | bool   | 是   | 新的参数值 |


```csharp
public void SetBool(int id, bool value);
```


| 参数名 | 类型   | 必要             | 说明       |
|:------ |:------ |:---------------- |:---------- |
| id     | int    | 是               | 参数 ID    |
| value  | bool   | 是               | 新的参数值 |
