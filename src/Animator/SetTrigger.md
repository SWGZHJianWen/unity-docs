# Animator.SetTrigger

设置触发器参数。

## 定义

```csharp
public void SetTrigger(string name);
```

```csharp
public void SetTrigger(int id);
```

## 参数

| 参数名 | 类型   | 必要             | 说明     |
| ------ | ------ | ---------------- | -------- |
| name   | string | 与 `id` 二选一   | 参数名称 |
| id     | int    | 与 `name` 二选一 | 参数 ID  |
