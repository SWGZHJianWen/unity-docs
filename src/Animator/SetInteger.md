# Animator.SetInteger

设置整型参数。

## 定义

```csharp
public void SetInteger(string name, int value);
```

| 参数名 | 类型   | 必要             | 说明       |
|:------ |:------ |:---------------- |:---------- |
| name   | string | 是              | 参数名称   |
| value  | int    | 是               | 新的参数值 |

```csharp
public void SetInteger(int id, int value);
```

| 参数名 | 类型   | 必要             | 说明       |
|:------ |:------ |:---------------- |:---------- |
| id     | int    | 是              | 参数 ID    |
| value  | int    | 是               | 新的参数值 |
