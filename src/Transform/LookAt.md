# Transform.LookAt

让物体面向某一物体

## 定义

```csharp
public void LookAt(Transform target, Vector3 worldUp = Vector3.up);
```

| 参数名  | 类型      | 必要 | 说明                              |
| ------- | --------- | ---- | --------------------------------- |
| target  | Transform | 是   | 指向的对象                        |
| worldUp | Vector3   | 否   | 指定向上方向的向量，默认世界 Y 轴 |
