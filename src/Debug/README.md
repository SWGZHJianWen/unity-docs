# Debug

包含帮助调试游戏的各种方法

| 方法                          | 说明                           |
| ----------------------------- | ------------------------------ |
| [Log](./Log.md)               | 将普通消息记录到控制台（白色） |
| [LogWarning](./LogWarning.md) | 将警告消息记录到控制台（黄色） |
| [LogError](./LogError.md)     | 将错误消息记录到控制台（红色） |
| [DrawLine](./DrawLine.md)     | 绘制一条直线                   |
